💡 왜 경고가 뜨나요?
Google Play Store에 없는 앱은 자동으로 경고가 표시됩니다. BROTHERHOOD는 탈중앙화 특성상 스토어 심사를 거치지 않아 이런 경고가 나타납니다.

2단계: 알 수 없는 소스 허용

Android에서 Play Store 외부 앱을 설치하려면:

  • Android 8.0 이상: 설정 → 보안 및 개인정보 보호 → 알 수 없는 소스에서 설치 → 브라우저/Chrome → 허용
  • Android 7.0 이하: 설정 → 보안 → 알 수 없는 소스 → 허용
  • MIUI (샤오미): 설정 → 추가 설정 → 개인정보보호 → 알 수 없는 소스 → 허용

3단계: APK 설치

다운로드가 완료되면:

  • 알림 창에서 "설치" 버튼 클릭
  • 또는 파일 관리자 → 다운로드 폴더 → APK 파일 터치
  • "앱 설치" → "설치" → "완료"
  • 경고 메시지가 뜨면 위의 해결 가이드 참고
